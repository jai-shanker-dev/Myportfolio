# 🌐 Personal Portfolio Website

A modern and responsive personal portfolio website built using **HTML**, **CSS**, and **JavaScript**.
This project showcases my skills, education, experience, services, and contact information with a clean and attractive user interface.

---

## ✨ Features

* Responsive design for all devices
* Modern and clean UI
* Multiple portfolio sections
* Contact form with email integration using FormSubmit
* Smooth navigation and layout
* Beginner-friendly project structure
* Easy to customize

---
